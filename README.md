How to contribute:
1. Clone the repo
   Run this:
   git clone https://github.com/logan03109/DDZ-game-jam.git
   cd DDZ-game-jam
2. Open the DDZ project - select the file "DDZ-game-jam"
3. If not in virtual environment, create it using:
   python -m venv .venv
4. Set the interpreter
   File → Settings → Project → Python Interpreter, click Add Interpreter → Add Local Interpreter. select Existing and point to .venv\Scripts\python.exe
5. Activate venv and install dependencies
   .venv\Scripts\activate
   pip install -r requirements.txt
6. Run the game using:
   python main.py
